# JavaScriptで文字列を反転する方法

本日のLeetCodeでの学び  
https://leetcode.com/problems/valid-palindrome/description/

JSには文字列を反転するメソッドが無い。配列を反転するメソッドはあるのでそれを利用する。

```js
const str = 'test'
const reversed = str.split('').reverse().join('');

console.log(reversed) // => 'tset'
```
